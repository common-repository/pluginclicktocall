== Description ==

ClickToCall

Ce plugin permet à vos internet de se faire appeler par téléphone en indiquant leur numéro de téléphone. 

Installer, configurer, insérer dans votre Sidebar, page, article. 

Les Services acceptés sont OVH, Orange, Ribbit, veuillez nous contacter pour enrichir cette liste.
